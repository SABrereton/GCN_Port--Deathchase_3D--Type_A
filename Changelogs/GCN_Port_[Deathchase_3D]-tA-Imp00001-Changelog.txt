=== Version 1 - [16-07-2026] ===

Initial version.

Consists of a fork from Version 0.9 of the game program "Deathchase 3D"
by Paul Robson.

This was obtained from an archived copy of Paul Robson's website
archived on 11th July 2007 by Internet Archive (https://archive.org/).

Notes, observations and/or changes follow:

    * Added a "Front Matter" component to the program from which users
      can set the parameters that they wish Deathchase 3D to use and
      also find information such as how to play Deathchase 3D and
      credits.

    * Renamed the main() function of Deathchase 3D to original_main()
      in-order to allow the program to begin in the new Front Matter
      using a main() function there and added declaration of the
      original_main() function prototype to dc.h.

    * Moved the initialisation of the components of the "Glo" instance
      of the "GLOBAL" struct to the main() function in
      GCN_Front_Matter.c instead of the GLOInitialise() function in
      globals.c.

      Access to them is needed in the GCN Front Matter for setting
      parameters which is first accessed prior to the use of
      GLOInitialise().

    * Added calls to "IOInitialise()" from original_main() when moving
      from and back to the Front Matter code in-order to reset the
      IO component(s) of Deathchase 3D.
      
      Deathchase 3D malfunctions without doing so.

    * Included a section at the beginning of each source code file
      which contains information about what program they are a
      component of.
